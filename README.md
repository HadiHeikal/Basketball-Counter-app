# Basketball Counter App

## Overview
The Basketball Counter App is a simple Flutter application designed to keep track of basketball scores. It provides an intuitive interface for users to increment scores for two teams, making it ideal for casual games or practice sessions.

## Features
- **Team Score Tracking**: Easily add points to each team's score.
- **Custom Widgets**: Reusable components for buttons and team columns.
- **State Management**: Utilizes the Cubit pattern for efficient state management.
- **Cross-Platform Support**: Runs on Android, iOS, Web, and Desktop platforms.

## Project Structure
The project is organized as follows:
- **lib/**: Contains the main application code.
  - **main.dart**: Entry point of the application.
  - **Cubit/**: Contains the Cubit and state classes for managing the app's state.
  - **Screens/**: Includes the main basketball app screen.
  - **widgets/**: Custom reusable widgets like buttons and team columns.
- **test/**: Contains widget tests for the app.

## Getting Started

### Prerequisites
- Flutter SDK installed on your machine.
- A code editor like Visual Studio Code or Android Studio.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/HadiHeikal/Basketball-Counter-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Basketball-Counter-app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```

### Running the App
To run the app on a connected device or emulator:
```bash
flutter run
```
